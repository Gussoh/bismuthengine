Compile:
 - Open the solution file in Project/BismuthEngine
 - Press the compile button
 
How to run:
 - Change the working directory of the TestApp project to '$(OutDir)\..\..\..\'
    * Right click on the project and choose properties
	* Go to the Debugging page
	* Set the working directory field
 - Press Ctrl + F5
 
 Note: You may need to download the latest DirectX runtime 
   http://www.microsoft.com/downloads/details.aspx?displaylang=en&FamilyID=2da43d38-db71-4c1b-bc6a-9b6652cd92a3
